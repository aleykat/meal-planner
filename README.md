# Weekly Meal Planner

A family meal planner: weekly menu with recipe cards, cost estimates, nutrition, and a
grocery checklist. Built around HEB weekly deals, sous vide + Blackstone cooking, and a
under-30-minutes-on-weeknights rule.

**Live site:** https://aleykat.github.io/meal-planner/

This public copy is the read-only view — menu, recipes, and the grocery checklist
(checkboxes persist per device). Feedback and pantry tracking run on a home server and
are hidden here.

Regenerated weekly.
